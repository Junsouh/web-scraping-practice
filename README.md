# Web Scraper

Run research_scraper.py to scrape website into directory html_files

Run analysis.py afterwards to print statistics/push dasta into a csv file titled scraping_analysis.csv

# CSV Files

scraping_analysis.csv contains all pages scraped, including some basic organization into categories and a character count

category_descriptions.csv contains the top words used in each description for every item

category_price.csv contains statistics regarding price (mean, max, min, titles of pages that are the max/min price)
